# Overview

Playground for CI Travis with node and express.

## Installation

First, clone this repo and change to the directory:

```bash
git clone git@github.com:pmtargosz/<project>.git
cd <project>
```

### Install

```bash
cd <project>
npm install
```

### Config

### Run

```bash
cd <project>
npm run dev
# http://localhost:3000
```
